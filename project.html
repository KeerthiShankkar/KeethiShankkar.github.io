CryptoPrice Tracker Extension

The CryptoPrice Tracker is a sleek and efficient Chrome extension designed to provide users with real-time cryptocurrency prices. By leveraging the power of WebSockets, this extension ensures instant and continuous updates, giving users the most accurate and up-to-date information on their favorite digital currencies.

Cypher Talks

Cypher Talks is a chatting application which was developed using Flutter framework . This is an application that has a special feature of disappearing messages on a set interval of time so the user need not worry about the privacy or the data leakage . All the user's data are stored in the Google Firebase
